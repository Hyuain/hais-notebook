## 页面导入样式时，`link` 和 `@import` 有什么区别？

| link | @import |
| --- | --- |
| HTML 标签，除了 CSS 之外还可以定义 RSS 等 | CSS 提供的 |
| 加载页面时同时加载 | 页面加载完之后再加载 |
| 没有兼容性问题 | 不兼容 IE 5 以下 |
| 可以通过 DOM 动态引入 | 不能动态引入 |
