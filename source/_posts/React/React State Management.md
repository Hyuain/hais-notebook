React 状态管理包含以下话题：

- [[useState|useState Hook]]：访问和管理 State 最简单直接的方式
- [[React Reducer]]：对 State 管理相关逻辑的进行抽象和封装
- [[React Context]]：使得数据可以跨层传递，而不必显式地通过组件树逐层传递 Props
- [[myRedux]]：结合 Reducer 和 Context 的简单状态管理方案
