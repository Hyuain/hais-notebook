Gem 和 Bundle

其他的 docker 命令

  • docker ps -a 查看所有的容器（Containers）
  • docker kill <id|name> 关闭对应的容器
  • docker restart <id|name> 重启关闭的容器
  • docer rm <id|name> 删除对应的容器
  • docker container prune 删除无用的容器，以节省空间
