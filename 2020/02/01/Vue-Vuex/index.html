基本使用

Vuex 的核心就是一个状态仓库，与普通的全局对象有所不同：

  1. Vuex 的状态存储是响应式的，store 变化 -> 组件响应式更新
  2. 不能直接修改 store 中的状态，只能通过提交 mutation 的方式来改变状态

State

Vuex 使用单一状态树，每个应用仅仅包含一个 store 实例
