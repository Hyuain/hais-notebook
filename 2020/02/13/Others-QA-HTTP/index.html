GET 和 POST 的区别？

  • GET 不安全，POST 不安全（其实都不安全）
  • GET 是有长度限制的，POST 没有长度限制（但一般 GET 的长度不超过 1 KB，POST 上限 4M ~ 20M 左右）
  • GET 参数在 URL 中，POST 消息在消息体中
  • GET 只需要一个报文，POST 需要两个以上（因为有消息体）
  • GET 幂等，POST 不幂等
  • 语义不同，GET 是为了获取数据，POST 是为了提交数据
