浏览器的渲染过程

根据 Google 团队的文章分析，我们可以将网页的渲染过程大致分为以下几步：

  1. 根据 HTML 构建 HTML 树（DOM）
  2. 根据 CSS 构建 CSS 树（CSSOM）
  3. 将两棵树合并为渲染树（Render Tree）
  4. 布局 Layout（文档流、盒模型、计算大小和位置）
  5. 绘制 Paint（边框颜色、文字颜色、阴影）
  6. 合成 Compose（根据层叠关系展示画面）
