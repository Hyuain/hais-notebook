关于堆栈内存和闭包

堆：存储引用类型值的空间；
栈：存储基本类型值和执行代码的环境；浏览器加载页面就会形成栈内存，当执行函数的时候，都会形成一个新的执行上下文（Execution Context Stack），压入栈中执行

闭包的作用 1：保存——保存私有变量，不被销毁

闭包的作用 2：保护——修改自己的私有变量，不会影响外面的东西
