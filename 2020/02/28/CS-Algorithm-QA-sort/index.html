算法小问题：排序

发表于 2020-02-28 分类于 计算机 ， 算法

介绍几种简单的排序方法。

可以看 这个网站 的动图

选择排序

每次从剩下的部分选择最小的排到前面来，O(n^2)

快速排序

找出 Pivot，把小于 Pivot 的放左边，大于 Pivot 的放右边，最好 O(nlogn)，最坏 O(n^2)

冒泡排序

两两比较，如果后者比前者小，则交换，最终每次循环使得最大数冒泡到最后去（O(n^2)，最好 O(n)）
